What is the difference between innerHTML, innerText, and textContent ?
Returns or sets HTML content inside an element, including HTML tags.
Returns or sets the visible text inside an element, considering CSS styles.
Returns or sets all text inside an element, ignoring CSS styles but keeping hidden text.
What is event delegation in the DOM?
Using a parent to handle events for child elements via event bubbling.
What is event bubbling in the DOM?
Events triggered on child elements propagate up to parent elements.
